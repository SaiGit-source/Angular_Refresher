When we make an HTTP request, it returns an observable because it is an Async process. it's like subscribing to newsletter, we get an update when there is new data available

We have injected HttpClient into CourseService and CourseService into CoursesList component

How do we get Service into a component? Dependency injection. 

constructor(private courseService: CourseService) => Angular creates a single instance or singleton of the courseService for the entire application

Cross-component communication: Different components need to communicate with each other, not necessarily directly connected. 
Database is one option, other is store a shared data in a service. 

if you want to share a wishlist between multiple components, we can add it to the CourseService. this pattern reduces the need for excessive @Input and @Output binding between components
